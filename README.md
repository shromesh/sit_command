# sit - simple tree-like command

### usage
1. Clone this repository
1. Set alias on .bash_profile or .zshrc
    ```
    alias sit=$HOME/sit_command/main.sh
    ```
1. Now you can use stree command!
    ```
    $ sit .
    ```
    ```
    .
    ./hoge
    ./hoge/fuga.txt
    ./hoge/fugafuga.txt
    ./hogefuga.txt
    ```

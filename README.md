# stree - simple tree command

### usage
1. Clone this repository
1. Set alias on .bash_profile or .zshrc
    ```
    alias st=$HOME/tree_command/main.sh
    ```
1. Now you can use stree command!
    ```
    sample $ st .
    >> .
    >> ./hoge
    >> ./hoge/fuga.txt
    >> ./hoge/fugafuga.txt
    >> ./fugahoge.txt
    ```
